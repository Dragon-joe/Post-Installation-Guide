# دليل ما بعد تثبيت ويندوز — النسخة التفاعلية

## التشغيل
افتح `index.html` مباشرة في أي متصفح حديث. لا يعتمد الموقع على PWA أو Service Worker.

## المميزات
- قائمة تجهيز تفاعلية مع نسبة إنجاز وحفظ تلقائي في `localStorage`.
- ست باقات: استخدام عادي، ألعاب، برمجة، تصميم ومونتاج، جهاز ضعيف، وطالب.
- أوصاف عربية مع الاحتفاظ باسم البرنامج الأصلي.
- صفحة تفاصيل لكل برنامج، ومفضلة، ومقارنات، وخطوات تجهيز وتحذيرات.
- نسخ أوامر WinGet للبرامج المدعومة.
- بحث وفلاتر وترتيب وعداد نتائج وزر رجوع للأعلى وشريط قراءة.
- SEO وOpen Graph وfavicon وصفحات الخصوصية والإبلاغ عن الروابط.
- **108 أيقونة SVG محلية مخصصة للبرامج** داخل `icons/apps/`، ولا تعتمد على CDN خارجي.

## بنية الأيقونات
كل برنامج يستخدم الملف:

`icons/apps/PROGRAM_ID.svg`

عند إضافة برنامج جديد إلى `assets/data.js`، أضف له أيقونة بنفس قيمة `id`.

## قبل النشر
استبدل `https://your-domain.com` بالدومين الحقيقي داخل ملفات HTML و`robots.txt` و`sitemap.xml`، أو شغّل:

`node tools/set-domain.mjs https://example.com`

## فحص الروابط
شغّل:

`node tools/check-links.mjs`

بعض المواقع تمنع الفحص الآلي، لذلك راجع أي نتيجة فاشلة يدويًا.

## المراجعة الأسبوعية للروابط
المشروع يتضمن Workflow داخل `.github/workflows/link-check.yml` لفحص الروابط دوريًا عند رفعه إلى GitHub.

## تحديث الواجهة ثنائية اللغة
- زر `EN / AR` في شريط التنقل يبدّل بين العربية والإنجليزية ويحفظ الاختيار داخل المتصفح.
- اتجاه الصفحة يتحول تلقائيًا بين `RTL` و`LTR`.
- تمت إضافة ترجمة إنجليزية لواجهة الموقع، صفحات التفاصيل، الأوصاف، المميزات، العيوب، التحذيرات، الباقات، والخطوات.
- تمت إضافة 7 أيقونات SVG مستقلة لباقات «اختر استخدامك الأساسي» داخل `icons/packages/`.
- تم تقوية الوضع الداكن للبطاقات، عناصر الإدخال، شريط التنقل، صفحات التفاصيل، المقارنات، والفيديوهات.
- تمت إضافة نظام أيقونات SVG للروابط والأزرار والفلاتر والتنبيهات والخطوات وصفحات التفاصيل.
- تمت إضافة خلفية متحركة خفيفة تدعم الوضعين النهاري والداكن وتتوقف تلقائيًا عند تفعيل تقليل الحركة في النظام.


## خريطة الموقع

- `sitemap.html`: خريطة مرئية للزوار تشمل الصفحات والباقات وكل البرامج.
- `sitemap.xml`: خريطة لمحركات البحث تشمل صفحات تفاصيل البرامج الديناميكية.
